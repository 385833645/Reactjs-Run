## Reactjs+ES6环境搭建入门

### 一、安装 [nodejs](https://nodejs.org/en/ "Title")

* 借助nodejs进行前端模块化项目构建，所以需要在开发环境先安装

### 二、cmd 到工程下（ReactjsES6）并输入：npm install

* 部分windows环境需要先执行 npm install webapck -g ，不然会出现无法找到命令的问题

### 三、执行打包命令：webpack

* 执行后生成可运行文件run.js

### 四、打开index.html即可看到运行结果
